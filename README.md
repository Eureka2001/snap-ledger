# SnapLedger

> 告别流水，看见变化。

## 简介

SnapLedger 是一个**完全离线**的、以资产现状盘点为核心逻辑的个人财务记录工具。完全抛弃财务记录软件中记录每笔账单的逻辑，而是以“快照”为核心逻辑，以用户自我盘点、自我复盘的内容为唯一输入数据，在用户的录入数据中进行分析。

## 核心特点

- **绝对离线** — 无需任何后端、无数据库，只要你有浏览器，双击 `index.html` 即可使用，数据存储在浏览器本地（IndexedDB）
- **隐私优先** — 数据加密存储（PBKDF2 + AES-GCM），忘记密码则数据不可恢复
- **数据主权** — 支持 CSV/JSON 导入导出，完全掌控个人数据
- **可视化分析** — 资产走势、收支对比、资产构成等多维度图表

## 使用方法

1. 双击 `index.html` 在浏览器中打开
2. 创建新账本，设置密码
3. 录入第一条快照数据（作为基准）
4. 后续定期录入快照，系统自动计算变化

## 技术栈

- 纯 HTML + CSS + JavaScript
- IndexedDB 本地存储
- ECharts 图表库（CDN）

## 数据安全

- 所有数据加密后存储在浏览器 IndexedDB
- 导出的文件为明文，请注意保管
- 建议定期导出备份
